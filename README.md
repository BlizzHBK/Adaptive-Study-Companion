# 🧠 Hindsight Learning Agent

An AI-powered study companion that learns from user mistakes, tracks weak topics, and adapts quizzes over time using memory.

---

## 🚀 What This Project Does

This app helps students improve by:

* Taking quizzes
* Tracking wrong answers
* Identifying weak topics
* Suggesting what to study next

Instead of forgetting past attempts, the system **remembers mistakes and adapts**, creating a smarter learning experience.

---

## 🧩 Features

* ✅ Quiz system with instant feedback
* ✅ Weak topic detection
* ✅ Persistent memory using JSON
* ✅ AI-like study insights
* ✅ Progress tracking
* ✅ Dynamic study plan

---

## 🧠 How It Works

```
User Answer → Evaluate → Store Mistakes → Identify Weak Topics → Suggest Improvements
```

* Wrong answers are saved
* Topics are tracked
* System highlights weak areas
* Study plan adapts accordingly

---

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON (memory storage)

---

## ▶️ How to Run

1. Install Streamlit:

```
pip install streamlit
```

2. Run the app:

```
streamlit run UI.py
```
